¿Qué es HTML semántico?

Echaremos un vistazo a la semántica HTML y por qué debemos usarlos

Introducción

Hoy en día, muchos de nosotros hemos utilizado HTML,ya sea porque estamos en el desarrollo web / software o simplemente por diversión. El uso de HTML es una cosa y puede parecer bastante fácil. Efectivamente, podemos obtener rápidamente algo simplemente jugando con HTML. Pero usar correctamente este lenguaje de marcado puede ser un poco más difícil de lo que parece.

¿Qué es HTML?

es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language

¿Por qué usarlo?

Entonces, ¿por qué deberíamos preocuparnos por HTML semántico? Teniendo en cuenta que las etiquetas HTML que utilizamos deben describir el significado subyacente de nuestro contenido, agregamos información adicional sobre nuestro documento y mejoramos nuestra comunicación.

El uso de etiquetas semánticas da las herramientas a los navegadores y a los diferentes motores de búsqueda para entender nuestro documento. Por lo tanto, obtenemos un contenido más que se puede buscar que, obviamente, conduce a una mejor clasificación de motores de búsqueda. También aumentamos la accesibilidad para dispositivos como lectores de pantalla que pueden interpretar más fácilmente el significado de nuestro contenido. Por lo tanto, los navegadores, motores de búsqueda o dispositivos pueden entender la estructura de nuestro documento y los roles de los elementos dentro de él.

El uso de etiquetas semánticas también puede darnos más ganchos para aplicar estilo a nuestro contenido.

Podemos pensar que este concepto fue introducido por HTML5,pero no es el caso. Existían muchas etiquetas semánticas antes de HTML5,pero esta última introdujo nuevos elementos.
